# DeskQuit

**DeskQuit** — это Windows‑приложение для контроля за физическим состоянием и здоровьем при длительной работе за компьютером. Оно отслеживает время вашей непрерывной активности за ПК и отправляет уведомления о необходимости сделать перерыв, размять глаза, встать из-за стола или выпить воды.

## 🌟 Ключевые возможности

- **Отслеживание активности (AFK System):** Умный таймер фиксирует только реальное время взаимодействия с ПК (отслеживает активность клавиатуры и мыши). Если вы отлучились за чаем, таймеры встают на паузу и не "спамят" уведомлениями по возвращении.
- **Плавающий таймер:** Виджет поверх всех окон, показывающий реальное время непрерывной сессии, мотивируя вас самостоятельно делать перерывы.
- **"Мягкие" уведомления:** Небольшие всплывающие окна для простых напоминаний (например, посмотреть вдаль на 20 секунд).
- **"Агрессивные" уведомления:** Блокирующие весь экран окна, которые заставляют оторваться от монитора и сделать полноценную физическую разминку.
- **Многоязычность:** Поддержка русского и английского языков "из коробки" с возможностью переключения прямо из трея.
- **Тихая работа в фоне:** Приложение может быть свернуто в системный трей, чтобы не отвлекать вас от основных задач.

## 🛠 Технологии

Проект разработан с использованием современных технологий и паттернов:
- **.NET 8.0 (Windows)**
- **Avalonia UI** (UI‑фреймворк для десктопных приложений)
- **MVVM** (Паттерн проектирования при поддержке `CommunityToolkit.Mvvm`)
- **DesktopNotifications.Avalonia** (Для системных уведомлений)
- **C# Win32 API** (Для отслеживания AFK состояния)

## 🚀 Установка и запуск

### Требования
- **Windows 10+** (x64).
- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).

### Запуск из исходников
1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/your-username/DeskQuit.git
   ```
2. Перейдите в папку с проектом:
   ```bash
   cd DeskQuit
   ```
3. Соберите и запустите приложение:
   ```bash
   dotnet run --project "DeskQuit/DeskQuit.csproj"
   ```

## 🏗 Архитектура проекта

- **`Views/`**: UI элементы, окна и их XAML разметка (Avalonia).
  - `Notifications/` - Окна для мягких и агрессивных уведомлений.
  - `FloatingTimerWindow` - Плавающий таймер непрерывной работы.
- **`ViewModels/`**: Логика представления для каждого окна.
- **`Services/`**: 
  - `NotificationService` - Ядро приложения, управляющее глобальным таймером и таймерами задач.
  - `UserActivityService` - Отслеживает время бездействия мыши и клавиатуры через нативные API.
  - `LocalizationService` - Отвечает за локализацию текстов приложения.
